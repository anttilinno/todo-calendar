## Milestone Summary

**Key Decisions:**
- VIS-checkbox-styling: Checkbox styled independently from text (accent [ ], green [x], strikethrough only on text)
- VIS-separator-width: Fixed 10-char Unicode box-drawing separator under headers
- HELP-short-keys: Show a/x/d/e// as the 5 most-used keys in short mode
- HELP-column-size: Group expanded bindings into columns of 5
- HELP-reset-on-tab: Reset ShowAll on pane switch to avoid stale expanded state
- EDIT-vertical-center: Edit form positioned at upper-third (topPad = (h-lines)/3)
- EDIT-single-field-edit: Edit title (e) and edit date (E) are separate single-field forms
- EDIT-remove-wsm: Todolist dimensions set via SetSize, not WindowSizeMsg
- EDIT-unified-confirm: Enter reads both title and date regardless of focused field
- EDIT-auto-focus-date: Empty/invalid date auto-switches focus to date field on Enter
- SEED-migration-v3: Seed templates via version-3 migration, not runtime count checks
- SEED-insert-ignore: Use INSERT OR IGNORE to handle existing user templates with same name
